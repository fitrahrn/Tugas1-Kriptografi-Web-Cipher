Fungsi visualisasi Model akan membuat nodes untuk bias dan layer input serta meng-assign lokasi koordinat nodes tersebut. Untuk semua layer selanjutnya, menggunakan loop yang akan membuat node-node dan kooridnati node tersebut untuk tiap layer serta membuat edges yang merepresentasikan weights dari node layer sebelumnya. Ia juga akan mencetak fungsi aktivasi yang digunakan untuk layer tersebut.
